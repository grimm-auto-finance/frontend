# frontend

## Usage

```sh
pnpm install # or npm, yarn
pnpm run dev # for development

pnpm run build # for production
pnpm run serve
```

See [tli-group-4-grimm/hosting](https://github.com/tli-group-4-grimm/hosting) for more details on usage in tandem with the backend via Docker or other means.
